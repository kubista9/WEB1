Try out the XML and AJAX. Retrieve and display external content on a web page.

NOTE: these exercises will only work if you use the Live Server extension in Visual Studio Code.

  1. In this session, you have to work in this html document and make the required changes in the JavaScript file included with the exercises.
  2. Create a new XML file that contains data about your SEP1 group members. You should at least have first name, last name, birthday, nationality, and email address, but you can include more information if you want to.
  3. Save the XML file in the xml folder that is included with the exercises.
  4. Add an attribute with the eye color of each group member in your XML file (attributes can't have spaces, so the attribute name should be eyeColor).
  5. Make a jQuery get function that gets your data from the xml folder. The callback function should find the first name of the first person in your file and dislpay it in the paragraph with the id "textField1".
  6. Modify your callback function so that it also displays the eye color of the first person in your file in the paragraph with the id "textField2".
  7. Create new paragraphs in this HTML file so that you can display the first name and eye color of all your SEP1 group members. Modify the callback function so that it will display all the data for all group members.
  8. Instead of displaying the data in paragraphs, you should now display the data in a table - you need to create the table as shown in the slides (as part of the txt variable on slide 28).
  9. Make a jQuery get function that gets data from the bookstore.xml file in the "xml" folder. The callback function should display the data in a table with the following table headings: Author, Title, Year. You should display the data according to the table headings and remember to give your table borders.
  10. Start working on creating a function to display the data from your SEP1 project on your website.
