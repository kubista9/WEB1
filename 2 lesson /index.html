Exercise, WEB1 session 2

Add external style-sheet to your Star Pizza website and make it pretty using CSS.

  1. Make a new folder for this session
  2. Copy your Star Pizza homepage into the new folder and work on this copy in today's session.
  3. Copy the ccs-folder along with the styles.css-file from itslearning into the (new)folder containing the Star Pizza homepage
  4. Add a link in the head section of the index.html for the Star Pizza homepage that links to the copied styles.css-file
  5. Change the font type and size of the headings (in the copied file)
  6. Add a link in the head section of the menu.html for the Star Pizza homepage to the styles.css-file, so both pages use the same styling
  7. Lower the font-size of the image captions
  8. Complete the learning path on itslearning (videos, links, and exercises).
