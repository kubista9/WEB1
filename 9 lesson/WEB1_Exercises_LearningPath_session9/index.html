Exercise, WEB1 Session 9 - Learning Path

You are not allowed to modify this HTML at all.

Exercise tasks

  1. Make functionality so when you click on one of the boxes it will disappear in either a sliding or fading effect.
  2. Make functionality so when you click the button here all boxes will (re)appear on screen.
  3. Add an event listener to the "document" object so when you press one of the arrow buttons, the blue box will move in that direction. Arrow key codes: left 37, up 38, right 39, down 40.
  4. Add delays into the squareMovement function which moves the orange box, so it waits one second at each corner before moving on.
  5. Add an event listener to the "Stop" button so the orange box stops its movement when the button is pressed.
  6. Add an event listener to the "Start" button so the orange box will start moving again when the button is pressed (just restart the movement - you don't have to "remember" where it stopped and all of that).
  7. Add an event listener to the green box so when you try to hover the moouse cursor over the box, the box will move to a random position within the browser window.
